我的第一個 Agent 任務

從頭到尾,實錄一次給你看

▶ 看影片 🎓 L2 上手 🎬 8 個段落 🀄 投影片 + 逐字稿
本集開場

歡迎進到實作篇的第一集。前面幾集我們把觀念講完了:什麼是 Agent、為什麼要交辦、跟聊天差在哪。道理你聽懂了,但我猜你心裡還是有一個疑問:講這麼多,實際做起來到底長什麼樣子?這一集就不講道理了,我挑一個最簡單的真實任務,從頭到尾完整做一次給你看。你會親眼看到一個 Agent,從我把任務交出去,到它自己做完、把結果交回來,中間的每一步。

1

觀念聽懂了,這集開始動手

🖐️
  • 1前面講的都是「為什麼」
  • 2這集開始講「怎麼做」
  • 3看一次,比聽十次都清楚
逐字稿

先講清楚這集的定位。前面觀念篇,我一直在跟你講「為什麼」:為什麼 Agent 不一樣、為什麼你該學交辦。但光聽道理,你很難真的有感覺。所以從這一集開始,我們切換到動手模式,講的是「怎麼做」。我自己的經驗是,這種東西看一次實際操作,比聽我講十次都還清楚。所以這集你就跟著我,看一個 Agent 任務完整跑一遍。

2

第一次,為什麼要挑最簡單的

🐣
  • 1不要一開始就挑複雜任務
  • 2先看懂那個循環:交辦 → 它自己做 → 交結果
  • 3循環看懂了,難的只是換內容
逐字稿

有件事我要先提醒你:第一次交辦,千萬不要挑一個很複雜的任務。我知道你可能很想馬上叫它做一件很厲害的事,但第一次的重點不是「做多難」,而是「看懂那個循環」。哪個循環?就是我把任務交辦出去、它自己動手做、最後把結果交回來,這一整圈。你只要先把這一圈看清楚,之後不管任務多難,其實都只是換一下裡面的內容而已,循環是一樣的。所以我故意挑一個很小、你一看就懂的任務。

3

我要交辦的任務

📋
  • 1我有一份很亂的清單:名字、電話、地址混在一起
  • 2我要它整理成一張乾淨的表格
  • 3夠簡單,你一眼就看得懂好不好
逐字稿

來看我今天要交辦的任務。我手上有一份很亂的清單,就是那種一行一行、名字、電話、地址全部黏在一起、格式還不統一的資料,看了頭很痛。我要做的事很單純:把它整理成一張乾淨、對齊的表格,一欄名字、一欄電話、一欄地址。我特意選這個,是因為它夠簡單,你一看就知道「整理好」長什麼樣子,也一眼就能判斷它做得對不對。這對第一次來說很重要。

4

我怎麼交辦

🗣️
  • 1把目標講清楚:我要一張表格
  • 2把要的結果講清楚:哪幾欄、怎麼排
  • 3剩下「怎麼做」,交給它想
逐字稿

接著是重點:我怎麼把這件事交辦出去。交辦不是隨便丟一句「幫我用一下」,我會把兩件事講清楚。第一,目標:我要一張整理好的表格。第二,我要的結果長什麼樣:我要三欄,名字、電話、地址,一行一筆。把這兩件講清楚就夠了,至於它要怎麼一步一步拆、怎麼對齊、怎麼處理格式不一致,那些「怎麼做」的部分,我不用管,交給它自己想。你會發現交辦的功夫,全在把目標和結果講明白。

5

接下來,實際做一次給你看

🎬
  • 1我把剛剛那份亂清單交給它
  • 2你會看到它自己一步步整理
  • 3看它怎麼從接到任務到交出表格
▶ DEMO(Tom 螢幕錄影):打開工具,把那份雜亂清單貼進去,用剛剛講的方式交辦「整理成名字/電話/地址三欄的表格」,然後不插手,讓它從頭做到尾,一路錄到它交出整理好的表格為止。
逐字稿

講了這麼多,不如直接看。接下來這一段,是我實際的螢幕操作。我會打開工具,把剛剛那份亂七八糟的清單交給它,用我剛剛說的方式把任務交辦出去,然後我就退到旁邊,不插手,看它自己一步一步整理。你會看到它從接到任務,到最後交出一張乾淨表格的完整過程。我們來看畫面。

6

剛剛發生了什麼

🔁
  • 1它自己把整件事做完了
  • 2我只做了兩件事:交辦、驗收
  • 3中間「動手」的部分,我完全沒碰
逐字稿

好,畫面看完了,我們倒回來看剛剛到底發生了什麼。你注意到沒有:整件事其實是它自己做完的。從頭到尾,我只做了兩件事,一件是最前面的「交辦」,把任務講清楚交出去;另一件是最後的「驗收」,看它給我的表格對不對。中間所有真正動手的部分,一筆一筆對齊、把格式喬好、排成表格,我完全沒有碰。這就是我一直在講的那個循環,你剛剛親眼看到它跑完一圈了。

7

換成舊做法呢

↔️
  • 1用問答:它只告訴我「整理清單的步驟」
  • 2然後我自己一筆一筆貼、一筆一筆對齊
  • 3同樣一件事,以前的手全是我在出
逐字稿

我們對比一下舊做法你更有感。如果我用以前那種問答的方式,我會問它「這種亂清單要怎麼整理」,它會給我一串步驟,告訴我先怎樣、再怎樣。然後呢?然後就要我自己動手了,自己一筆一筆貼、一筆一筆對齊、自己排成表格。同樣一件事,以前那些出力的手,全部都是我在出。你把這兩種擺在一起看,交辦跟問答的差別,是不是一下子就具體了。

8

這集帶走一句話

💡
  • 1第一次交辦,重點不是做多難
  • 2是看懂「交辦 → 它做 → 驗收」這個循環
  • 3循環熟了,任務自然越交越大
下集:交辦要怎麼講,它才會做對?目標 + 驗收標準,兩個都要給。
逐字稿

這一集你帶走一句話就好:第一次交辦,重點從來不是做多難的事,而是看懂「交辦、它做、驗收」這個循環。這一圈你熟了,之後任務自然就會越交越大、越交越複雜。不過你剛剛可能也發現了,我今天交辦講得特別清楚,它才做得這麼順。那萬一講不清楚會怎樣?交辦到底要講到什麼程度,它才會做對?下一集,我就專門講這件事:一個好的交辦,要給它目標,還要給它驗收標準,兩個都不能少。我們下集見。
