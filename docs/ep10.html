複雜任務:它自己規劃、執行、還會驗證

交辦的事,可以更大了

▶ 看影片 🎓 L3 進階 🎬 7 個段落 🀄 投影片 + 逐字稿
本集開場

大家好,歡迎來到進階篇第十集。前面幾集我們交辦給 Agent 的任務,其實都還算單純,大多是一步就能做完的事。這一集我要帶你升級,交辦一種需要「規劃」的任務,一種很多步驟、一步接一步的複雜任務。你會看到它怎麼自己把大事拆開、照順序做下去,而且做完還會回頭檢查對不對。

1

複雜任務難在哪

🧩
  • 1步驟很多,不是一步就做完
  • 2步驟之間有先後、有依賴
  • 3中間錯一步,後面全歪掉
逐字稿

先講清楚複雜任務難在哪。它不是一步就能解決的,而是有很多步。更麻煩的是,這些步驟之間有先後順序、有依賴關係,後面那步要用到前面那步的結果。所以只要中間錯了一步,後面全部都會跟著歪掉。這種任務,你以前大概不太敢整個丟給 AI,因為你會怕它做到一半就亂了。

2

Agent 怎麼處理:先規劃

🗺️
  • 1拿到大任務,先不急著動手
  • 2把它拆成一小步一小步
  • 3排出該先做什麼、後做什麼
逐字稿

那 Agent 是怎麼處理這種任務的?第一步,它會先規劃。拿到一個大任務,它不會急著動手,而是先想清楚:這件事可以拆成哪幾個小步驟?哪一步要先做、哪一步要後做?就像一個做事有條理的人,會先在腦袋裡列出步驟,再開始動工。這個「先想再做」,就是它能扛住複雜任務的關鍵。

3

Agent 怎麼處理:依序執行、邊做邊驗

⚙️
  • 1照著計畫,一步一步做
  • 2每做完一步,回頭檢查對不對
  • 3錯了就退回修,不會一路錯到底
逐字稿

規劃好之後,第二步就是照著計畫一步一步做下去。而且它不是悶著頭做到底,是邊做邊檢查。每完成一步,它會回頭看一下這步的結果對不對、能不能接到下一步。如果發現不對,它會退回去修,而不是帶著錯誤一路衝到終點。規劃、執行、再加上驗證,這三件事湊起來,它才有辦法把複雜任務做完。

4

實際看它做一個多步驟任務

🎬
  • 1我丟一個真的很多步的任務給它
  • 2看它自己規劃、一步步做
  • 3還會自我檢查,錯了回頭修
▶ DEMO(Tom 螢幕錄影):我丟一個有好幾個相依步驟的真實任務給 Agent(例如「上網查資料 → 整理成表 → 做分析 → 產出一份報告」這種一步接一步的任務),全程看它先規劃、再依序執行、中間自我檢查、發現錯了回頭修,最後產出成品。
逐字稿

講到這裡,不如直接看它做。接下來我丟一個真正多步驟的任務給它,一個要先上網查資料、再整理、再分析、最後產出一份報告的任務,每一步都要接著上一步。我們一起看它怎麼自己規劃、怎麼一步一步做下去,還會自己回頭檢查。我們來看畫面。

5

剛剛發生了什麼

🔁
  • 1它不是一次就吐出答案
  • 2是先想步驟,再一步步做
  • 3做完還會回頭檢查,像人一樣
逐字稿

看完了。回頭看它剛剛做的事:它不是我一問就一次吐出一整包答案。它是先想好步驟、排好順序,再一步一步做下去,而且做完還會回頭檢查對不對。這個過程,其實跟一個做事仔細的人很像:先想清楚再動手,做完再核對一遍。這已經不是「回答問題」,而是「完成一件事」。

6

跟單步任務差在哪

📈
  • 1單步:一問一答就結束
  • 2複雜任務:它自己撐完整段流程
  • 3你能交辦的事,一下子變大了
逐字稿

這跟前面單步的任務差在哪?單步任務就是一問一答,一下就結束了。而複雜任務,它能自己撐完一整段流程,從頭走到尾。這個差別對你來說很重要,因為它意味著,你能交辦出去的事情,一下子變大了。以前只敢丟小事給它,現在整個專案裡的一大段,你都可以放心交出去。

7

這集帶走一句話

💡
  • 1複雜任務靠的是「規劃 + 驗證」
  • 2不是更聰明的一次回答
下集:一次派很多個 Agent 同時做事
逐字稿

如果這一集你只帶走一句話,那就是:Agent 能搞定複雜任務,靠的不是「更聰明的一次回答」,而是「會規劃、會驗證」這兩件事。它把大事拆小、排好順序做、做完再檢查,靠的是流程,不是靈光一閃。那如果一個 Agent 就能撐完整段流程,一次派很多個 Agent 同時做事會怎樣?下一集,我們就來玩這個。我們下一集見。
