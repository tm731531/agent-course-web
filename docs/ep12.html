我實際怎麼用 Agent 處理真實工作

而最真的例子,就是你正在看的這一切

▶ 看影片 🎓 L3 進階 🎬 10 個段落 🀄 投影片 + 逐字稿
本集開場

這一集,我不講教科書例子,我把我自己真的在用 Agent 的方式,打開給你看。而最真的例子,我不用去別的地方找,就是你正在看的這整個系列。這一整套影片,就是我用 Agent 做出來的。我來一步一步拆給你看。

1

先講一個有點難堪的事實

😅
  • 1我不會剪影片
  • 2我不想對著麥克風配音
  • 3我沒有做投影片的設計手感
逐字稿

先講一個有點難堪的事實。我不會剪影片,我也不想對著麥克風一句一句錄旁白,我更沒有把投影片做得好看的那種設計手感。照理說,一個同時有這三個缺點的人,是做不出一整門影片課的。這三件事,隨便一件都能把整個計畫卡死。

2

以前的結局:要嘛不做,要嘛花大錢

🧱
  • 1自己學剪輯、配音、設計 → 幾個月
  • 2外包 → 一大筆錢 + 來回溝通
  • 3多數人的結局:算了,不做了
逐字稿

如果是以前,我大概只能這樣:第一,自己去學剪輯、學配音、學設計,少說幾個月,而且不一定學得會。第二,花錢外包,十九集的剪輯加配音,輕鬆就是幾萬塊,還要一直來回溝通。而大部分人走到這一步,結局就是四個字:算了,不做了。這種課,通常就死在這裡。

3

現在:我把「執行」整段交辦出去

🤝
  • 1我出的:方向、主題、每個決定
  • 2它出的:腳本、投影片、配音、字幕、上傳
  • 3一句話:我出腦,它出手
逐字稿

但這一次不一樣。我把「執行」這一整段,交辦給了 Agent。我負責的是方向、是這個系列要講什麼、是每個岔路口的決定。它負責的,是把腳本寫出來、把投影片生出來、把旁白合成出來、把字幕上好、再把影片傳上去。分工非常清楚:我出腦,它出手。

4

連 demo 都是真的 agent 跑的

🤖
  • 1不是演的,也不是截圖拼的
  • 2真的叫一個 agent 去做真任務
  • 3把它的操作過程自動擷取下來
逐字稿

最誇張的是,你前面幾集看到的那些 demo,不是我演出來的,也不是把截圖拼起來的。是真的叫一個 agent 去跑一個真的任務,像是把一份雜亂的名單整理成乾淨的表格,然後把它實際的操作過程,自動擷取下來變成畫面。連示範,都是真的在做,不是在演。

5

直接秀給你看:這門課的產線

⚙️
  • 1一個指令:生投影片 + 合成配音
  • 2一個指令:真 agent 跑出 demo
  • 3一個指令:自動上傳 YouTube
▶ DEMO:這門課的產線實際在跑(自動生成)
逐字稿

光用講的你可能沒感覺,所以我直接把產線秀給你看。一個指令,生出一集的投影片跟配音;另一個指令,讓真的 agent 跑出一段 demo;再一個指令,把影片自動傳上 YouTube。整條線跑起來,我只是在旁邊看、需要的時候調一下。你接下來看到的,就是它真的在動。

6

數字:省的不只是時間

📊
  • 119 集,在一個工作階段裡做完
  • 2傳統做法約 150 小時體力活
  • 3還跳過我根本不會的技能
逐字稿

給你一個真實的數字。這十九集,是在一個工作階段裡做出來的。如果用傳統方式,一支影片抓八到十五小時,十九支就是大概一百五十個小時的體力活,晚上抽空做,要好幾個月。但比時間更關鍵的是:它跳過了我根本不會、也不想學的技能。那些本來會直接把我卡死的牆,直接不存在了。

7

但我沒有把大腦關掉

🧠
  • 1每一集我都一支支看過
  • 2抓到一集旁白跟畫面對不上
  • 3退回去修好,才留著
逐字稿

但我要特別強調一件事:我沒有因為它自動,就把大腦關掉。這些影片,我是一支一支看過的。中間我還真的抓到一集,旁白講的數字,跟畫面上的條列對不上,我就把它退回去、修好、再重傳一次。這一步非常重要,因為如果我沒看,那個錯就這樣公開出去了。

8

這正好就是這系列的核心

⚖️
  • 1執行,可以交辦
  • 2判斷,不要外包
  • 3越自動,越要有人把關
逐字稿

你有沒有發現,這整件事,剛好就是這個系列從頭到尾在講的那句話:執行可以交辦,判斷不要外包。越是自動、越是快,那個「有沒有人在把關」就越重要。我省掉的,是我不會的、重複的體力活;我留下來的,是只有我能做的判斷跟驗收。這兩件事的分工,就是整套方法的重點。

9

你也可以,從一件事開始

🌱
  • 1挑一件你重複做、又討厭做的工作
  • 2先練習把它講清楚、交辦出去
  • 3做順一件,再慢慢擴大
逐字稿

所以你也可以。你不用一次就做一整門課,先挑一件你每天重複做、又有點討厭做的工作,練習把它講清楚、交辦給 Agent,把一件事做順了,再慢慢擴大。你會發現,很多你以為「我不會,所以我做不了」的事情,其實是「我不會,但我可以交辦」。這中間的差別,大到會改變你能做的事的範圍。

10

這集帶走一句話

💡
  • 1你剛看完的一切,本身就是證明
  • 2不會的技能,交辦掉
  • 3該你扛的判斷,留下來
下一步:把你自己的第一個任務,交辦出去
逐字稿

如果這一集你只帶走一句話,那就是這句:你剛剛看完的這一整個系列,本身就是證明。一個不會剪片、不會配音、也沒有設計手感的人,靠著把執行交辦出去、把判斷留在自己手上,做出了一整門課。不會的技能,交辦掉;該你扛的判斷,留下來。這,就是從「問 AI」到「用 Agent」,真正的差別。
