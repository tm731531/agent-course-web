「聊天」和「交辦」差在哪

很多人以為「我在跟 ChatGPT 對話,那不就是 Agent 嗎?」——差別不在介面,在誰在做事。下面這個,你可以自己點點看。

▶ 看影片

🖐 自己試試看:同一件事,兩種做法

任務:比較三家廠商報價,挑出最划算的一家
→
聊天:你問一句、它答一句,上下文和下一步,都是「你」在補。做事的是你。
→
交辦:你給一個目標,它自己把中間步驟補齊、自己動手、自己檢查,直接把結果交給你。做事的是它,你負責驗收。
→
會這樣「交辦」的 AI,才叫 Agent。它要能:有目標、會用工具、會自己迭代修正(下一集拆解)。
